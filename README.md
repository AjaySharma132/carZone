# CarZone
A used car selling website where a buyer can look for a car according to his needs, ask queries to seller, filter cars etc. Itâ€™s main functionalities are - admin panel customization, search functionality, pagination, authentication with Google and Facebook and sending emails using django.

# Installation Process
1. Fork this repository
2. Set up a new virtual environment
3. Clone this repository
4. Change into the project directory and install the project dependency by running `pip install -r requirements.txt`
5. Run the project locally by `python manage.py runserver`.
